学生管理

在学生管理当中可以管理学生的所有信息, 包括地址、父母、课程等.同时提供学生学习当中的调课、请假等，可以查看学生作业情况.

教师管理

教师管理与系统的HR模块集成, 方便的对教师进行管理，能够从教育背景、个人信息、工资等进行管理

科目管理

科目管理用于管理科目相关的信息设置 比如科目名称、科目分类等.

课程管理

课程管理允许设置课程相关的详尽信息. 允许设定课程的类型,并且允许对课程设定关了的课程.

排约课计划

根据课程和教师资源情况制定排课计划或约课计划. 如果是约课计划还允许学生按照开放的教师时间进行自主约课.
